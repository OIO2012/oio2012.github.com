Summary

You have now learned the basics of using Emacs. If you feel that Emacs is difficult to use, due to the different functions for doing one thing (such as moving around), please stick to the simple functions and when you get used to these, learn the other functions.

In this hour, you saw some powerful things, such as reading compressed files. See how many other editors are capable of doing this! If you are totally new to Emacs, please read these sections without trying to remember everything. They provide a sense of what Emacs can do. Later, when you need to use these features, you can return and read this hour more carefully.
